Detailed Description

Unit providing ping pong delay processing of streo sample stream.

ALERT: Whole class requires correct usage as written in documentation. It does not make any argument checks nor it throws any own error,

float PingPongDelay::PingPongDelayUnit::GetPanoramaParam ( )

Gets the panorama parameter setting of unit.

Returns:
panorama parameter between [0, 1]. In case zero is returned unit is delaying firstly all delayed signal as mono to the left, then to the right channel. In case of one half is returned unit puts all the delayings as stereo to the center each time. In case of one is returned unit is delaying firstly all delayed signal as mono to the right, then to the left channel. Any other settings in between these act evenly corresponding to the value.

float PingPongDelay::PingPongDelayUnit::primaryPanningQuotient_ [private]

primaryPanningQuotient_ & secondaryPanningQuotient_ are used to provide stereo delaying. They represent the ratio of respectively left and right channel input sample to be delayed in left channel and the ratio of respectively right and left channel input sample to be delayed in right channel. They are always complementary. Their sum must be equal to 1.
